# 🧁 Olsker Cupcakes – Fullstack Webshop

A fullstack cupcake webshop that was built with Java, Javalin, Thymeleaf and PostgreSQL as part of a school project.
Users can build custom cupcakes, manage a cart, place orders as well as view order history, where an admin can simply view all orders.

---

## Features

### User Features

* Register and login
* Build custom cupcakes (bottom + topping)
* Live price calculation
* Add items to cart
* Session-based cart (with persistence)
* Checkout flow
* View personal order history
* Logout functionality

### Admin Features

* View all orders in the system
* Separate admin UI

---

## Tech Stack

### Backend

* Java
* Javalin
* JDBC
* PostgreSQL

### Frontend

* Thymeleaf (server-side rendering)
* HTML + CSS
* JavaScript 

---


## Setup Instructions

### 1. Clone repository

```
git clone https://github.com/Eligoon/Cupcake.git
```

### 2. Run the application

Run the `Main` class.

Server starts at:

```
http://localhost:7070
```
